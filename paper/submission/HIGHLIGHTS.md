# Highlights (3–5 bullets, ≤ 85 characters each)

- A hardware-first, four-layer taxonomy of KV cache compression for LLM inference.
- First KV-cache survey to consolidate multi-tenant cache security and side channels.
- First to analyse KV cache degradation in long-horizon agentic reasoning loops.
- Introduces Matched-Budget Evaluation (MBE): a fixed-budget protocol + open harness.
- Co-design analysis of how stacked compression methods interfere, with a Pareto view.
